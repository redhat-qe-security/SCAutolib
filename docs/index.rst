Installation
************
.. note::
   SCAutolib is designed for RPM based Linux distribution like RHEL 8, CentOS 8,
   Fedora 30 (or later versions of mentioned distributions) because of specific
   RPM packages used for smart cards testing like SSSD and Authselect.

.. code-block:: bash

   $ pip install SCAutolib
